The Employee Payroll System is a simple Java-based console application that demonstrates the implementation of Object-Oriented Programming (OOP) concepts such as:

Abstraction Inheritance Polymorphism Encapsulation

Collection Framework (ArrayList) The system allows you to: Add employees Remove employees Calculate salaries dynamically Display employee details

Employee Types: 🔹 Full-Time Employee Has fixed monthly salary. Salary = Monthly Salary 🔹 Part-Time Employee Paid based on hours worked. Salary = hoursWorked × hourlyRate
